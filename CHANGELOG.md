## 0.1.0 - 2026-05-04
### Added
* New `agent-quota` CLI tracks AI provider OAuth subscription quotas — pretty TUI for humans, headless JSON for scripts and agents
* New `claude` provider tracks Claude Code subscription quotas — 5-hour, weekly, and weekly-Sonnet windows
* New `copilot` provider shows GitHub Copilot usage and premium request quotas
* New `gemini` provider tracks Gemini Code Assist daily quotas via the `gemini` CLI's OAuth credentials
* New `openai` provider tracks ChatGPT Codex usage
* New TUI dashboard renders provider usage as scrollable cards with budget guide markers
* `--json` flag and `aq status` produce one-shot JSON output for scripts and agents
* New `aq` short alias — installer symlinks `aq` → `agent-quota`
* New `aq self-update` — atomic updates with sha256 verification (Linux/amd64)
* New `scripts/install.sh` one-line installer for prebuilt Linux/amd64 releases
* Persistent dashboard settings save shown providers, card order, header visibility, and refresh interval
* Dashboard auto-refreshes on a configurable interval — `Ctrl+R` forces immediate refresh

> Update with `aq self-update` on Linux/amd64.
