## 0.3.3 - 2026-04-18
### Fixed
* tui: guide marker now stays inside the filled region when utilization > guide. At narrow bar widths the fill cell count and marker cell were rounded independently, which could place the │ past the visual fill edge (e.g. util=20%, guide=16% on a 12-cell bar rendered as `██│░░░░░░░░░`, contradicting the numeric ordering). A new `guidePosition` helper snaps the marker inward so visual ordering matches numeric ordering. (#60)

## 0.3.2 - 2026-04-17
### Added
* Minimal `.devcontainer/` (Go 1.25 + gh CLI) and `scripts/e2e-smoke.sh` smoke test that builds the CLI and exercises the `--help` / `--version` wiring. Catches regressions like the self-update permission-denied bug that no unit test would see.
### Fixed
* self-update: ensure the installed binary keeps its executable bit. `os.CreateTemp` creates the staging file with mode 0o600, and `O_CREATE|O_TRUNC` does not re-apply `perm` on an existing file, so the atomically-renamed binary ended up non-executable. `copyFile` now explicitly `Chmod`s to the requested mode.

## 0.3.1 - 2026-04-17
### Fixed
* self-update: point release lookup at rudolfjs/agent-quota (was schnetlerr/agent-quota, which returned 'No releases found'). Also fixes scripts/install.sh default OWNER_REPO.

## 0.3.0 - 2026-04-17
### Added
* Added 'agent-quota self-update' (aka 'aq self-update') which downloads the latest GitHub release, verifies the sha256 checksum, and atomically swaps the running binary. Supports --check (report only), --force (reinstall same version), and --pre (include prereleases). Linux/amd64 only.
### Fixed
* Claude provider: normalize extra usage cents when used_credits carries sub-cent fractions from per-token billing, so the TUI shows dollar amounts instead of raw cent values
* Normalize Claude extra_usage cents→dollars for non-USD accounts (e.g. AUD), which previously leaked raw cent values into the TUI.

## 0.1.1 - 2026-04-10

## 0.1.0 - 2026-03-30
### Added
* New `changie` workflow manages unreleased fragments and release-note generation
* New GitHub CI runs `lefthook`, lint, tests, and builds for pull requests
* New GitHub Releases build cross-platform binaries and ship a `scripts/install.sh` installer
* `agent-quota` dashboard auto-refreshes on a configurable interval
* `agent-quota` dashboard scrolls through provider cards with a visible scrollbar
* New `make` targets simplify local build, test, and release checks
* New persistent dashboard settings save shown providers, card order, header visibility, and refresh interval
* New `copilot` provider shows GitHub Copilot usage and premium request quotas
### Changed
* Dashboard now supports manual refresh with `Ctrl+R` and a 5-minute refresh option
* GitHub Releases now verify tags point to `main` before publishing binaries
